# Season of Grind

A web application that visualizes gym workout statistics in an interactive wrapped-style presentation. Displays comprehensive fitness data including workout frequency, exercise rankings, volume analysis, heatmaps, and peak performance metrics.

## Tech Stack

- React 18.3
- TypeScript 5.8
- Vite 5.4
- Tailwind CSS 3.4
- Shadcn UI
- React Router DOM
- TanStack Query
- Recharts

## Getting Started

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:8080`

### Build

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

- `src/components/slides/` - Individual slide components
- `src/context/` - React Context providers (Audio, Data)
- `src/data/` - Workout data and mappings
- `src/pages/` - Page components

## Requirements

- Node.js (v18 or higher)
- npm (v9 or higher)
